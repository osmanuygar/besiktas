# Besiktas

Besiktas CLI!

## Installation

```shell
$ pip install besiktas
```

## Usage
Main Page
```shell summary
$ besiktas
```
![](docs/main.png)
List of Players
```shell footballers
$ besiktas -kadro
```
![](docs/kadro.png)
List of Top Scorers
```shell footballers
$ besiktas -stats
```
![](docs/stats.png)
