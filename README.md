# Besiktas

Besiktas CLI! Aq biraz kosun!

## Installation on local

```shell
$ pip install besiktas
```
## Installation with Docker

```shell
$ docker run -ti osmanuygar/besiktas
```


## Usage
Main Page
```shell summary
$ besiktas
```
![](docs/main.png)
List of Players
```shell footballers
$ besiktas -kadro
```
![](docs/kadro.png)
List of Top Scorers
```shell footballers
$ besiktas -stats
```
![](docs/stats.png)
